vim 总结
===============================

操作技巧
-------------------

1. 全局粘贴: ``"+y``(复制) ``"+p``(粘贴)
2. 输入多个相同字符:  ``80i=<Esc>`` (连续输入80个=)
3. 自动匹配单词: ``<C-p>``(向前搜索可匹配的) ``<C-n>`` (向后搜索可匹配的)
4. 匹配当前目录文件名 ``<C-x C-f>`` (插入状态下, 搜索可匹配的文件名并完成)
5. 跳转到匹配的括号: ``%``
6. 跳转到最后一次修改处: `` `. `` (左上角+句号)
7. 将所有tab转换为空格: `` :retab ``
